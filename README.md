Hair Apparent is a fully functional, modern barbershop website built for a Newport-based shop serving students, professionals, and military personnel in the City by the Sea. The site combines clean design with interactive features to make booking a haircut simple, fast, and engaging.

✂️ Overview

Hair Apparent allows customers to:

Explore popular haircut styles (fades, crew cuts, buzz cuts, beard trims, and more)

Learn about services and pricing

Book appointments online

Use an interactive haircut recommender tool

Navigate a responsive, mobile-friendly interface

The goal of the project is to create a polished, real-world product website that blends functionality with personality — just like a great neighborhood barbershop should.

🚀 Features
🔹 Interactive Haircut Recommender

A decision-tree powered tool that evaluates user preferences and suggests a haircut style based on:

Length preference

Maintenance level

Professional vs. casual look

This feature uses conditional logic and dynamic UI updates to enhance user experience.

🔹 Online Appointment Booking

Customers can:

Select services

Choose preferred haircut type

Submit booking details

The form includes input validation and user feedback to prevent errors like double submissions.

🔹 Dynamic UI Elements

JavaScript-powered interactivity

DOM manipulation

Conditional rendering

Loop-based content updates
